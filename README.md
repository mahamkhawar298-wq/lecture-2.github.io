# My Story · Maham

A semantic HTML personal profile page for Maham from Gujrat, showcasing her love for movies.

## Overview

Single-page profile built with semantic HTML5 and custom CSS. Features include personal story, favorite movies list, and contact section — all in a soft beige color scheme.

## Tech Stack

- HTML (semantic elements: header, nav, section, article, figure, details, blockquote)
- CSS

## Quick Start

Open `index.html` in any browser.
